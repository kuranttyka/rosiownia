# rosiownia.com

Personal website of **Rouzeris** — artist, jewelry maker, and AI app developer.

## Tech Stack

- [Astro](https://astro.build) — static site framework
- [React](https://react.dev) — interactive components
- [Tailwind CSS v4](https://tailwindcss.com) — styling
- TypeScript

## Features

- Dark, calm "Midnight Atelier" color palette
- Animated canvas cat (Yuma the Maine Coon) that follows your cursor
- Smooth minimal design
- Responsive layout

## Development

```bash
npm install
npm run dev
```
